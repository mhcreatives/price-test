⚡ Quick Start (3 steps)

1

Install Tampermonkey

Tampermonkey is a free browser extension that lets you run custom scripts on websites. Install it for your browser:

Get Tampermonkey
2

Install the Script

Open the script file below in Tampermonkey to install it. Tampermonkey will prompt you to confirm.

📜 Install Script

Or open Tampermonkey → Dashboard → New Script, paste the contents of shopee-checkout.user.js, and save.

3

Go to Shopee Checkout

Add items to your cart, proceed to checkout, and the automation panel will appear in the bottom-right corner of the page automatically.

Configure your preferences in the panel and click ▶ Run Automation.

✨ Features

📍 Address Selection

Auto-selects your preferred shipping address by matching a keyword (e.g. "Makati", "Home", your street name).

💳 Payment Method

Automatically picks your preferred payment — GCash, ShopeePay, COD, credit card, or any listed option.

🏷️ Voucher / Promo Code

Enters and applies a voucher or promo code automatically before checkout.

⏱️ Auto Place Order Optional

Places the order after a configurable countdown (0–30 s). Useful for flash sales — disable it to review manually.

All settings are saved in your browser automatically — you only need to configure once.

🔥 Flash Sale Tips

To maximize your chances during a Shopee flash sale:

  1. Add items to cart before the sale starts.
  2. Open the checkout page and let the script pre-fill address & payment.
  3. Enable Auto-place order with a countdown of 0–3 seconds.
  4. Click ▶ Run a few seconds before the sale starts — the countdown will fire exactly when you set it.
  5. Keep the tab focused and your internet stable.
Note: Shopee may update its page structure. If a step fails, check the browser console (F12 → Console) for [Shopee Auto] messages to diagnose the issue.

❓ FAQ

Does this work on shopee.ph?
Yes. The script is configured for shopee.ph. You can edit the @match lines in the script header to support other regional Shopee domains (e.g. shopee.sg, shopee.com.my).
Is my data safe?
All settings are stored locally in your browser via Tampermonkey's storage — nothing is sent externally. The script only runs on Shopee checkout pages.
The automation panel doesn't appear.
Make sure Tampermonkey is enabled and the script is active (green dot in the Tampermonkey icon). Try refreshing the checkout page after the cart is loaded.
A step says "not found" in the console.
Shopee occasionally changes its HTML structure. Open an issue or update the CSS selectors in the script to match the current page layout.
Can I disable just one step?
Yes — each feature has its own toggle. Uncheck the ones you don't need.
