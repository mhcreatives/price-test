📱 Android Tablet — Recommended Method

Install Firefox for Android (free on Play Store), then add Tampermonkey. This is the easiest and most reliable method.

1

Install Firefox for Android

Open the Google Play Store and install Firefox (free).

Firefox on Android supports real browser extensions — unlike Chrome for Android.
2

Install Tampermonkey in Firefox

In Firefox, open the menu → Add-ons → search for Tampermonkey → Install.

3

Install the Automation Script

Tap the button below to open the script file. Tampermonkey will prompt you to install it.

📜 Install Script
4

Go to Shopee Checkout

Add items to cart, go to checkout — the automation panel appears at the bottom of the screen automatically.

🔖 Alternative: Bookmarklet (any Android browser)

Works in Chrome, Samsung Internet, or any browser — no extensions needed. Save it as a bookmark, then tap it on the Shopee checkout page.

1

Add the bookmarklet to your browser

Tap Copy below, then:

  1. Open any website in your browser
  2. Bookmark that page (Star icon)
  3. Go to Bookmarks, find the one you just saved, tap Edit
  4. Delete the URL and paste the copied code
  5. Save — name it Shopee Auto
2

Use it on checkout

Go to shopee.ph/checkout, open your bookmarks, and tap Shopee Auto. The panel will appear.

🍎 iPad — Recommended Method (Safari)

Install the free UserScripts Safari extension from the App Store, then add the script to it.

1

Install UserScripts from App Store

Search for "UserScripts" by Justin Wasack on the App Store (free).

2

Enable it in Safari Settings

Go to Settings → Safari → Extensions → enable UserScripts. Grant access to shopee.ph.

3

Add the script

Tap the script button below to open the .user.js file. UserScripts will offer to install it automatically.

📜 Install Script
4

Go to Shopee Checkout

Proceed to checkout on Shopee — the automation panel appears in the corner.

🔖 Alternative: Bookmarklet (no app needed)

Copy the code below, then create a Safari bookmark with it as the URL.

1

Create the bookmarklet

  1. Tap Copy below
  2. In Safari, bookmark any page (Share → Add Bookmark)
  3. Open Bookmarks, tap Edit on the new bookmark
  4. Replace the URL with the copied code
  5. Save — name it Shopee Auto
2

Tap it on checkout

Safari may ask "Are you sure you want to run a script?" — tap Run to confirm.

✨ Features

📍 Auto Address

Picks your preferred delivery address by keyword.

💳 Auto Payment

Selects GCash, ShopeePay, COD, or any method by name.

🏷️ Auto Voucher

Enters and applies your promo/voucher code.

⏱️ Auto Place Order

Places order after a 0–30 s countdown. Great for flash sales.

⚙️ Configuration

SettingDescriptionExample
Address keywordPartial text in your saved addressMakati, Home
Payment keywordName of your preferred paymentGCash, COD
Voucher codePromo or discount codeSAVE50
CountdownSeconds before auto-placing order5

Settings are saved in your browser — configure once, works every time.

🔥 Flash Sale Tips

  1. Add items to cart before the sale starts
  2. Open checkout and configure address + payment in the panel
  3. Enable Auto-place order with a 0–3 s countdown
  4. Tap ▶ Run a few seconds before the sale — the countdown fires at exactly the right moment
  5. Keep the screen awake and your internet connection stable

❓ FAQ

The panel doesn't appear.
Make sure the extension/userscript is enabled. Refresh the checkout page after your cart is fully loaded.
A step says "not found".
Shopee sometimes updates its page layout. Try running again; if it keeps failing, the CSS selectors in the script may need updating.
Can I skip a step?
Yes — each feature has its own toggle. Turn off the ones you don't need.
Is my data safe?
All settings are stored locally in your browser. Nothing is sent anywhere outside Shopee.
